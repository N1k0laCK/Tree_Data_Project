# Binary Search Tree Project

## Description
This project implements a Binary Search Tree (BST) as part of a 12th grade Informatics assignment.
The program allows creating a tree, adding elements, searching for elements, and traversing the tree.

## Tree Type
Binary Search Tree (BST)

## Technologies
- Programming language: C#
- Platform: .NET Console Application

## How to Run
1. Open the project in Visual Studio or any C# IDE
2. Run the program
3. Use the menu to interact with the tree

## Example
Input:
Add: 5, 3, 7, 1

Output:
Traversal: 1 3 5 7
